#### Stage 1-B: 건축가능영역 (Build-line) 산출

```python
# 의사코드
def calculate_buildable_area(parcel, zone_type, road_info):
    setback = get_setback_distances(zone_type)
    buildable = parcel.buffer(-setback)
    if has_district_plan(parcel):
        buildable = buildable.intersection(designated_line)
    if is_corner_lot(parcel, road_info):
        buildable = buildable.difference(corner_cutoff)
    fire_route = calculate_fire_access_route(parcel, road_info)
    buildable = buildable.difference(fire_route.buffer(3))
    return buildable
```

#### Stage 1-C: Max Envelope + 매스 최적화

```python
# 의사코드
def generate_max_envelope(buildable_2d, regs, north_angle):
    envelope = extrude(buildable_2d, height=regs.max_height)
    if regs.sunlight_restriction:
        envelope = boolean_diff(envelope, sunlight_plane(north_angle))
    if has_adjacent_buildings():
        envelope = boolean_diff(envelope, daylight_planes())
    envelope = clip_to_far_limit(envelope, regs.max_far)
    return envelope
```

## 10. AI 대화형 인터페이스 설계

### 10.1 대화 시나리오 예시

```
사용자: "경남 김해시 주촌면 농소리 631-2 대지를 분석해줘"
AI: 📍 대지 정보를 불러왔습니다.
    • 면적: 10,623㎡  • 용도: 학교용지
    • 건폐율: 60% 이하  • 용적률: 180% 이하
    최적 배치안을 생성할까요?

사용자: "교육연구시설(특수학교) 5층 이하, 25학급 기준으로 배치해줘"
AI: 🏗️ 5개 배치안을 생성했습니다.
    [안①] 남향 ㄱ자형 — 용적률 172%, 일조 최우수
    [안②] 중정형 — 용적률 165%, 동선 최적
    [안③] ㄷ자형 — 용적률 178%, 학급수 최대

사용자: "안③으로 하되, 체육관을 동측에 별동으로 배치해줘"
AI: ✅ 체육관을 동측 별동으로 분리 배치했습니다.
    ⚠️ 소방차 진입로가 4.8m로 축소 (기준: 6m)
    → 2m 남쪽 이동으로 해결 가능. 적용할까요?
```

### 10.2 LLM 연동 구조

```
사용자 자연어 → LLM → Intent 분류 + 파라미터 추출
├─ "배치 수정" → 배치 엔진 호출
├─ "법규 확인" → 법규 엔진 조회
├─ "환경 분석" → 시뮬레이션 실행
├─ "사업성" → 재무 분석 호출
└─ "일반 질문" → 법규 RAG 검색
→ 결과를 3D 뷰어 + 대시보드에 반영 → 자연어 응답 생성
```

### 15.3 프로세스별 상세 구현 계획

#### P1. 법규적 가이드라인 기반 볼륨 검토

| 항목 | 구현 방법 | 데이터 소스 |
